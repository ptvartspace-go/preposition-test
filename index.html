How to use
For students: open any worksheet and start with the Reference tab to read the grammar notes. Then work through the activities in order — each one builds on the last.
  • Click a word chip, then click a blank to place it.
  • Answers are checked immediately — green means correct, red means try again.
  • Japanese translations appear below each question to support understanding.

For teachers: each activity ends with a teacher tip for follow-up speaking or discussion tasks. Use the worksheets as a starting point, not a finishing line.
